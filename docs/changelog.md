# SGCL — Changelog

## 0.1.0-alpha

Data: 25/06/2026

### Adicionado

- Backend Node.js com TypeScript
- Prisma ORM
- SQLite em desenvolvimento
- Autenticação JWT
- Middleware de autenticação
- Middleware de autorização por perfil
- Cadastro de usuários
- Controle de usuário ativo/inativo
- Dashboard
- Cadastro de alunos
- Edição de alunos
- Inativação de alunos
- Ficha completa do aluno
- Frontend React com Vite
- Login no frontend
- Layout principal
- Página de alunos
- Página de detalhes do aluno
- Design System inicial

### Componentes criados

- Button
- Input
- Card
- Layout
- ErrorMessage
- PageHeader
- StatusBadge

### Decisões importantes

- Backend e frontend separados
- Faixa do aluno não é alterada pelo cadastro
- Usuários inativos são bloqueados
- Design System será construído antes da expansão de módulos
